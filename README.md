# Supply Chain Analytics Dashboard

## Project Overview
This project is an interactive Supply Chain Analytics Dashboard developed using Tableau Public. The dashboard provides insights into sales performance, inventory management, manufacturing efficiency, product quality, and logistics operations.

## Dashboards
1. Executive Overview
2. Sales & Inventory
3. Manufacturing & Quality
4. Supply Chain & Logistics

## Tools Used
- Tableau Public
- Microsoft Excel

## Key Metrics
- Revenue Generated
- Production Volume
- Manufacturing Cost
- Defect Rate
- Products Sold
- Stock Levels
- Shipping Cost
- Supplier Lead Time
